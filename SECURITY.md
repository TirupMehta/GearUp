# Security Policy

## Supported Versions

Currently, only the latest release of ShopGenie AI is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.0.1  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of ShopGenie AI seriously. If you discover a security vulnerability, we'd appreciate it if you could bring it to our attention so we can fix it.

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, please report them to **4c10sanidhya66322@gmail.com**.

We will try to acknowledge your email within 48 hours. If the vulnerability is accepted, we will coordinate with you to outline a timeline for the fix and public disclosure.

We appreciate your effort and responsible disclosure and will make every effort to acknowledge your contributions.
